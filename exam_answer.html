MongoDB — Installation, Atlas Setup, CRUD (1-page)

Install MongoDB (Windows quick)

  1. Download Community Server (MSI): https://www.mongodb.com/try/download/community and run the installer.
  2. Install as a Service (optional), optionally install Compass.
  3. Install mongosh: https://www.mongodb.com/try/download/shell, verify mongosh --version.

Atlas (cloud)

  1. Sign up at Atlas → Create a Free Cluster (M0).
  2. Create DB user (username/password) in Security → Database Access.
  3. Allow your IP in Network Access (for tests you can add 0.0.0.0/0, but don't do this for production).
  4. Use the connection string (mongodb+srv://...).

Node quick connect

const { MongoClient } = require('mongodb');
const client = new MongoClient(process.env.MONGO_URI || 'mongodb://localhost:27017');
await client.connect();
const db = client.db('myDB');
await db.collection('students').insertOne({ name:'A' });
await client.close();

CRUD shell commands

// Insert
db.students.insertOne({ roll_no:1, name:'Alice', department:'CSE', marks:85 })
// Find
db.students.find({ department:'CSE' }).pretty()
// Update
db.students.updateOne({ roll_no:2 }, { $set:{ marks:82 } })
// Delete
db.students.deleteOne({ roll_no:4 })

Fast demo (no local MongoDB)

npm run start:memory
# Starts an in-memory MongoDB and runs CRUD demo (uses mongodb-memory-server)
